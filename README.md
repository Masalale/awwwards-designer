## Design Philosophy

### The Hadnoir Principles

1. **Mathematical Intentionality** â€” Every value is purposeful
2. **Bespoke Creation** â€” Never reuse aesthetic categories
3. **Motion as Meaning** â€” Animations communicate purpose
4. **Framework Agnosticism** â€” Design transcends implementation

### Forbidden Patterns (Auto-Reject)

The skill explicitly prevents:
- Inter/Roboto as display fonts
- 3-column feature grids
- Purple gradient on white ("AI slop")
- `transition: all`
- Linear easing
- Images without dimensions
