# Awwwards Skill Suite

A system of five coordinated Claude Code skills for building award-quality websites — from brand brief to shipped product. Design is generated by Google Stitch MCP. Animation is derived from brand metaphor, not borrowed from templates.

Targets Awwwards SOTD, SOTM, and SOTY.

---

## The Pipeline

```
Brief Interview  →  Invention Gate  →  Stitch MCP  →  Enhancement Layer  →  Quality Gate
```

| Phase | What Happens |
|-------|-------------|
| **Brief Interview** | 6 questions. Brand identity, audience, animation tier, pages, interactions, references. |
| **Invention Gate** | Brand metaphor extracted → motion language defined → animation map produced → Stitch prompt written. No code yet. |
| **Stitch MCP** | Google Stitch generates all screens. `build_site` outputs a TanStack Start project. `DESIGN.md` captures the full Design DNA. |
| **Enhancement Layer** | Animation and interaction layer built on top of Stitch output. `DESIGN.md` is the single source of truth — nothing overrides it. |
| **Quality Gate** | 8-gate pass/fail checklist. Ship only when every gate passes. |

## Skills

| Skill | Role |
|-------|------|
| [`awwwards-designer`](./awwwards-designer/) | Orchestrator. Drives all 5 phases. |
| [`awwwards-animation-standards`](./awwwards-animation-standards/) | Easing vocabulary, tier matrix, GPU animation rules, scroll, text, and transition patterns. |
| [`awwwards-performance-standards`](./awwwards-performance-standards/) | Core Web Vitals targets, JS bundle budgets, LCP/CLS/INP optimization, image and font strategies. |
| [`awwwards-quality-gate`](./awwwards-quality-gate/) | Pre-launch checklist: Visual · Motion · Responsive · Accessibility · Performance · Interaction · Technical. |
| [`awwwards-visual-standards`](./awwwards-visual-standards/) | Typography, color, Lucide icons, spacing, layout grid — deferred to `DESIGN.md` after Phase 3. |

The orchestrator drives the workflow. The four standards skills are autonomous knowledge bases — a Claude agent selects them based on context.

## Install

```bash
git clone https://github.com/Masalale/awwwards-designer.git
cd awwwards-designer
bash install.sh
```

Copies all five skills to `~/.agents/skills/` where Claude Code discovers them automatically.

**Individual skill:**
```bash
bunx skills add https://github.com/Masalale/awwwards-designer --skill awwwards-designer
```

---

## Hard Stops

These are non-negotiable. Any violation is a build failure.

- **TanStack Start only** — No Astro, Next.js, or any other framework
- **Bun only** — No npm, npx, or yarn
- **Lucide icons only** — One system, one visual language
- **`100dvh` not `100vh`** — Mobile address bar accounts for
- **No touch scroll override** — Native touch momentum is sacred
- **Descender safety on display text >48px** — `padding-bottom: 0.18em` minimum
- **Context7 before any library integration** — Training data is stale; APIs change
